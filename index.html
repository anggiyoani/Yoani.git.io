ANSWER NO2 UAS => Cara kerja sebuah website melibatkan beberapa elemen utama, termasuk web server, domain, dan hosting. Berikut adalah penjelasan singkat tentang hubungan antara ketiganya: Web Server: Web server adalah perangkat lunak atau perangkat keras yang menyimpan, mengelola, dan mengirimkan konten website kepada pengguna melalui internet. Ketika seseorang mengakses website, permintaan dikirimkan ke web server untuk mendapatkan halaman atau data yang diminta. Domain: Domain adalah alamat unik yang digunakan untuk mengidentifikasi lokasi suatu website di internet. Domain membantu pengguna mengakses website tanpa perlu mengingat alamat IP yang panjang. Hosting adalah layanan penyimpanan dan pemeliharaan file dan data website di server yang terhubung ke internet. Ketika seseorang mengakses website, data dari website tersebut diambil dari server hosting dan dikirimkan ke perangkat pengguna melalui internet. Hosting dapat dikelola sendiri (self-hosting) atau disediakan oleh penyedia layanan hosting yang mengelola server untuk Anda. Hubungan antara Web Server, Domain, dan Hosting: Domain diarahkan ke alamat IP server hosting menggunakan DNS. Ketika seseorang memasukkan domain ke dalam peramban web, sistem DNS akan mencari alamat IP yang terkait dengan domain tersebut.
